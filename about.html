📘 QA Test Management Platform

⬅ Kembali ke Dashboard

Platform web ringan untuk membantu tim QA mengelola test case, requirement, pelaksanaan testing, visualisasi hasil, dan pelaporan secara terpusat dan fleksibel.

🧰 Fitur Utama

  • Test Case Management: CRUD test case, penjadwalan, status, label, versi, log, navigasi, dan export CSV/XLSX/PDF.
  • Requirement Management: Form lengkap dengan user flow, device, performance, recovery, serta upload diagram dan export PDF.
  • Reporting & Visualisasi: Pie/Bar chart, AI Suggestions & Summary, matrix coverage, progress, export chart ke PDF.
  • Setup Proyek: Form setup dengan rangkuman, link, gambar, dan export ke PDF.

🧪 Teknologi

Stack teknologi ringan berbasis:

  • HTML, CSS, JavaScript
  • Tailwind CSS untuk styling responsif
  • Chart.js, jsPDF, SheetJS
  • Penyimpanan data via localStorage (tanpa backend)

🚀 Cara Menjalankan

  1. Clone repository ini: git clone ...
  2. Buka index.html di browser
  3. Login/Register, lalu mulai membuat project

⚠️ Semua data tersimpan di browser menggunakan localStorage.

📬 Rencana Integrasi

  • Pengiriman laporan via Email/Slack/Odoo
  • Integrasi ke pipeline CI/CD
  • Penggabungan laporan test + setup + requirement
